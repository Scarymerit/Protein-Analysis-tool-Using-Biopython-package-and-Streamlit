OverviewThis pipeline takes a raw PDB file (Protein Data Bank) as input and automatically performs a full bioinformatic analysis. It bridges the gap between structural biology and sequence analysis by extracting sequences directly from 3D models and mapping conserved motifs back onto the 3D structure.Perfect for: Bioinformatics students, Structural Biologists, and Researchers needing quick visualization of protein families.‚ú® Key Featuresüì• PDB Extraction: Automatically extracts peptide sequences from PDB structure files.Aignment (MSA): Performs Reference-based Multiple Sequence Alignment using global pairwise alignment.üîç Motif Discovery:Generates WebLogo visualizations.Identifies conserved residues across chains.Plots motif frequency.üå≥ Phylogenetics: Constructs and saves UPGMA and Neighbor-Joining (NJ) trees (Newick + PNG format).üß™ 3D Visualization:Motif Mapping: Highlights conserved motifs in Red on the 3D structure.Hydrophobicity: Highlights hydrophobic residues (A, I, L, M, F, W, V) in Orange.üìä Database Fetching: (Optional) Fetches metadata from GenBank and SwissProt.‚öôÔ∏è InstallationTo run this pipeline, you need Python installed along with the following libraries.1.
